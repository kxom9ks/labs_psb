# Script SSH

Первоначально нужно изменить содержимое в файле /etc/hosts FQDN и имя машины, после только включается скрипт со сменой имени машины и генерацией ключей с дальнейшей отправкой хостам

Пользование:

На клиентских машинах ./s.sh Client1 (создаётся пользователь ssh_Client1)

На серверных машинах ./s.sh Server ssh_Client1@192.168.x.x(создаётся пользователь ssh_Server, создаются пара приватный и публичный ключи, переаётся публчиный ключ сервера на клиентские машины)
